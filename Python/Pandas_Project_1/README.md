# Pandas Data Merging Project

## Описание
В этом проекте я тренировался работать с библиотекой **Pandas** для анализа данных. 

## Что было сделано:
* Загрузка данных из формата `.csv` и `.xlsx`.
* Очистка данных и обработка пропусков.
* Объединение нескольких таблиц (Merging) для получения общего датасета.
* Базовая визуализация результатов.

## Инструменты
* **Python 3.11**
* **JupyterLab**
* **Pandas**
